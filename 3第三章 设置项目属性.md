# 📱 iOS 项目属性设置指南

在 Xcode 中配置项目基础属性，是开发的第一步。以下是关键设置项详解：

---

## 🎯 Supported Destinations（支持的平台）
指定 App 可运行的设备类型：
- iPhone
- iPad
- Mac（需 Catalyst 支持）

> 💡 通常选择 **iPhone & iPad** 即可。

![设置项目属性](images/设置项目属性.png)

---

## 📏 Minimum Deployments（最低部署版本）
- 设置 App 支持的 **最低 iOS 版本**（如 iOS 15.0）
- 影响可用 API 范围

> ⚠️ 建议：根据用户分布选择（目前主流为 iOS 15+）

---

## 🪪 Identity（应用身份标识）

| 设置项 | 说明 |
|-------|------|
| **App Category** | 应用商店分类（如“教育”、“工具”） |
| **Display Name** | 用户看到的 App 名称（可含中文） |
| **Bundle Identifier** | 全球唯一标识，格式：`com.公司名.App名` |
| **Version** | 对外发布的版本号（如 `1.0.0`） |
| **Build** | 内部编译号（每次提交 TestFlight 需递增） |

---

## 📐 Deployment Info（部署信息）

### 设备方向支持
- Portrait（竖屏）
- Landscape Left/Right（横屏）

### Status Bar Style（状态栏风格）
- **Light**：浅色文字（适合深色背景）
- **Dark**：深色文字（适合浅色背景）

> ✨ 小技巧：可在代码中动态切换：
> ```swift
> override var preferredStatusBarStyle: UIStatusBarStyle {
>     return .lightContent
> }
> ```

---

## 🖼️ App Icons and Launch Screen（图标与启动页）

### 启动画面配置
| 选项 | 说明 |
|------|------|
| **Launch Screen File** | 指定启动界面文件 |
| `LaunchScreen.storyboard` | 推荐！自适应布局，支持动态字体 |
| `Main.storyboard` | 不推荐！会延迟主界面加载 |

![设置启动界面](images/设置启动界面.png)

> 🔍 **最佳实践**：  
> - 使用 `LaunchScreen.storyboard`  
> - 添加 Auto Layout 约束  
> - 避免复杂动画（启动速度优先）
