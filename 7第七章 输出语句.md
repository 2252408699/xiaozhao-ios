# 📱 iOS 输出语句

```swift

// 基础打印
print("hello world")

// 调试打印（带格式、更适合调试）
debugPrint("it") //可根据内容类型打印相应格式的内容

// Core Foundation 类型打印
CFShow("It" as CFTypeRef) // 打印 CF 类型对象，C字符串转 CF 类型指针

// CF = 苹果底层 C 语言核心工具库
// Swift / OC 都是基于它建起来的

// 详细镜像打印（输出结构+类型）
let position = (x:10.5,y:20)
dump(position) // 输出对象的详细内容

```
