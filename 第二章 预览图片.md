## 📚  《ios 预览图片》

- Assets **资源文件夹**，导入图片素材，添加一张图片

![资源文件夹](images/资源文件夹.png) 

- ViewController **试图控制器文件** 打开

```swift
import UIKit

class ViewController: UIViewController {

    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        
        let image = UIImage(named: "adSense") //创建图像类，用来夹在和绘制图像
        let imageView = UIImageView(image: image) // 将图片指定给图片视图，可看做图片的容器
        imageView.frame = CGRect(x:35,y:80,width: 320,height: 568) //设置图像显示区域 坐标(35,80), 宽高(320，568)
        self.view.addSubview(imageView) //图像视图 添加到 当前视图控制器的根视图
    }
}

```
